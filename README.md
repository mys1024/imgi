# imgi

A CLI tool that prints images' information.

## Usage

```bash
imgi [DIR] # default DIR is "./"
```

**imgi** prints images' information in **YAML** format:

```txt
images:
  - name: 1.jpg
    width: 2160
    height: 2880
  - name: 2.jpg
    width: 2160
    height: 2880
  - name: 3.jpg
    width: 2160
    height: 2880
  - name: 4.jpg
    width: 2160
    height: 2880
```

## Support image

- JPEG (.jpg, .jpeg)

- PNG (.png)

- WEBP (.webp)

- GIF (.gif)
